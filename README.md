# Restaurant Website Frontend
Сайт включает следующие страницы:
- Главная страница
- Меню ресторана
- Галерея блюд
- Блог и новости
- Информация о ресторане
- Контактная информация
#Технологии
- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (ванильный)
- Адаптивный дизайн
